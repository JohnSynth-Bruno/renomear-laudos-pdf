# Renomeador Automático de Laudos em PDF 📄➡️👤

Este é um script em Python desenvolvido para automatizar o fluxo de digitalização de clínicas e laboratórios. Ele lê PDFs escaneados, identifica o nome do paciente no documento usando Regex e renomeia o arquivo automaticamente.

## 🚀 Problema que resolve
Impressoras e scanners salvam documentos com nomes genéricos (ex: `Scan_001.pdf`). Este script elimina a necessidade de abrir arquivo por arquivo para renomeá-los manualmente com o nome do paciente, economizando horas de trabalho administrativo.

## 🛠️ Tecnologias Utilizadas
* Python 3.x
* `pdfplumber` (Extração de texto de PDFs)
* `re` (Expressões Regulares para identificar o nome)
